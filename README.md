# 🏦 Financial System Automation (Python & Playwright)

### 📊 Layihənin Məqsədi
Bu layihə maliyyə sistemlərində valyuta məzənnələrinin həm istifadəçi interfeysi (UI), həm də daxili servis (API) səviyyəsində avtomatlaşdırılmış testlərini əhatə edir. 

### 🧪 Test Ssenariləri
1. **UI Validasiyası:** Valyuta çeviricisi üzərində dinamik axtarış və nəticələrin əks olunması.
2. **API Validasiyası:** Backend-dən gələn JSON datanın strukturunun və valyuta cütlüklərinin yoxlanılması.

### 🛠️ Texnologiyalar
- **Language:** Python 3.x
- **Framework:** Playwright (Python)
- **Test Runner:** Pytest
